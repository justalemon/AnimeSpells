## Prerequisites

* GTA V 1.0.1604.0 (After Hours) or higher
* [ScriptHookV](http://www.dev-c.com/gtav/scripthookv/) 1.0.1604.1 or higher
* [ScriptHookVDotNet](https://github.com/crosire/scripthookvdotnet/releases) 2.10.9 or higher
